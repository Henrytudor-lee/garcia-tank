## Design Context

### Users
- Casual gamers looking for nostalgic arcade experience
- Retro game enthusiasts
- Players who enjoy classic Battle City tank game

### Brand Personality
- Retro arcade aesthetic
- Nostalgic yet polished
- Fun and energetic
- 80s arcade machine vibe

### Aesthetic Direction
- **Neon Retro Style**: Neon light effects, dark backgrounds with glowing elements, paying homage to 80s arcades
- Dark theme with vibrant neon accent colors
- Glowing/CRT-style effects for text and UI elements
- Classic tank game color palette retained (green player, red/orange/purple/blue enemies)

### Design Principles
1. Dark backgrounds with neon accent colors create arcade atmosphere
2. Use glow effects sparingly for emphasis (titles, important buttons)
3. Maintain the classic tank color identity (green=player, red=enemy)
4. Score/level indicators should be prominent with high contrast
5. Keep UI elements clean and readable despite the neon aesthetic

### Color Palette
- **Background**: Deep black (#000000) with subtle dark blue/purple tints
- **Primary (Player)**: Neon Green (#00ff00)
- **Secondary (Multiplayer)**: Neon Yellow (#ffff00)
- **Accent**: Cyan (#00ffff), Magenta (#ff00ff)
- **Danger/Enemy**: Neon Red (#ff0000), Orange (#ff8800)
- **UI Elements**: Dark gray with neon borders

### Application Strategy
- Title: Yellow with red glow effect (already has text-shadow)
- Player indicators: Green (P1), Yellow (P2)
- Enemy indicators: Use enemy type colors
- Score display: High contrast cyan or white
- Buttons: Dark with neon border glow on hover
- Overlays: Semi-transparent dark with neon accents
