# 坦克大战 - 测试文档

## 项目概述

本项目是一个经典坦克大战游戏的 Web 复刻版，使用 Next.js + TypeScript 开发，支持用户注册登录、自定义地图创建、排行榜等功能。

### 技术栈

- **前端框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **数据库**: Supabase (PostgreSQL)
- **认证**: Supabase Auth
- **测试工具**: Playwright
